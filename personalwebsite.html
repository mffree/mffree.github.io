About this website

I wanted to create a personalized online presence that was not constrained by the structures of existing social media.

I have built a few websites over the years but--before getting started with this venture--first had to brush up on my HTML and CSS.

After buying a domain name, I set about coding the homepage in Sublime Text.

Once I had something basic, I explored different ways to host it. Github Pages seemed like the best choice, and using Github would afford me the chance to learn more about modern software deployment using the system that is ubiquitous in industry. I've submitted all updates to this website through commits to my Github repository.

I am continuing to add more content to my website as I come up with ideas. I like how it looks so far. Making this website has been delightful and eye-opening.
